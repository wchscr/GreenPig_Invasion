## 绿皮猪入侵

## 1 介绍

一款基于pygame开发的2D游戏！

需要安装pygame包，步骤如下

```
pip install pygame
```

运行主文件alien_invasion.py开始游戏！

## 2 游戏操作

飞船的移动：方向键`<- ->`进行左右移动；

子弹的发射：`sqace`键发射普通子弹，`z`键发射超级子弹；

退出游戏：按`q`键退出游戏。
